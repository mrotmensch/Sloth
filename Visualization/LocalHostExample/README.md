## Dependencies

# Python

```sh
pip install -r requirements.txt
```

# JavaScript

Node.js
```sh
curl -sL https://deb.nodesource.com/setup | sudo bash -
sudo apt-get install nodejs
sudo ln -s /usr/bin/nodejs /usr/bin/node
```
Bower
```sh
npm install -g bower
```
Bootstrap, jquery, D3
```sh
bower install bootstrap#3.2.0 jquery#2.1.1 d3#3.4.11 --save
```
